# ソートアルゴリズムの実装

## 実行方法
python main.py

## 実装したアルゴリズム

本課題では以下の2つを実装しました。

- 挿入ソート (Insertion Sort)
- マージソート (Merge Sort)

## 実装のポイントと設計の意図

### 挿入ソート

最初は要素を抜き出して正しい位置に挿入する方法で考えていましたが、Pythonのスワップ構文を使うことで、隣の要素と比較しながら左へずらしていくシンプルな実装に落ち着きました。

**好きな理由**

手札を並べ替えるときの感覚に近く、アルゴリズムの動きが頭の中でイメージしやすいところが気に入っています。
また、ほぼ整列済みのデータに対して効率よく動作する点も、
「状況によって最適な手段は変わる」という考え方につながり、実務的な視点でも興味深いと感じました。

### マージソート

**好きな理由**

大きな問題を最小単位まで分解して、一つずつ解決してから組み合わせるという流れがシンプルで理解しやすいと感じています。
また、どんなデータでも計算量がO(n log n)で安定している点と、同じ値の順序が変わらない「安定ソート」である点は、実務での信頼性という意味で重要だと学びました。

**実装で意識した点**

- `pop(0)` はデータ量が増えるほど処理が重くなるため、
  `collections.deque` の `popleft()` を採用しました
- 安定ソートの特性を保つため、マージ時の比較条件を慎重に設定しました
